# Tools Runtime 设计（子文档，占位）

本文档将细化工具运行时：工具集合定义、权限合并规则、审批机制、审计格式、以及脚本执行的受控策略（超时/工作目录隔离/环境变量清理/输出截断）。

## 1. 工具集合（建议最小集）

- 只读：read_file/list_dir/grep
- 执行：run_script
- 破坏性：write_file/delete_file/network_request（默认关闭）

## 2. 权限模型

- 最终允许工具 = 全局允许 ∩ 技能 allowed-tools ∩ 运行时策略
- 高风险工具：强制审批与可重放命令输出

## 3. 审计

- 结构化记录：动作、参数、结果摘要、错误、耗时、退出码
